## в—Ҳ Example `.fam` File

```python
App(
    appid          = "my_app",
    name           = "My App",
    apptype        = FlipperAppType.EXTERNAL,
    entry_point    = "my_app_main",       # вҶҗ must match your C function name exactly
    requires       = ["gui"],
    stack_size     = 2 * 1024,
    fap_category   = "Misc",
    fap_description= "My cool app",
    fap_author     = "YourName",
    fap_version    = "1.0",
)
```

> [!IMPORTANT]
> `entry_point` must match your function name in your `.c` file **exactly** вҖ” case sensitive.
